Receiving OSC Data from TouchOSC
Version: 1.0	

Function:
OSC Data Communication to Control AC 220V Lighting

Description:
Create Control Panel using TouchOSC Editor
Assign Arduino to handle OSC Data coming from Ethernet to control Lighting System
 
Hardware Configuration:
Ethernet Shield: D10,D11,D12,D13,D50,D51,D52,D53
Pin 2 [DIG]  : Output AC Lighting 1
Pin 3 [DIG]  : Output AC Lighting 2
Pin 4 [DIG]  : Output AC Lighting 3
Pin 5 [DIG]  : Output AC Lighting 4
